# ProductManagementSystem

**Project Goals**

This project was developed to strengthen my full-stack software development skills by building a real-world inventory management application. It demonstrates experience in frontend development, backend API design, database management, CRUD operations, and version control using Git and GitHub.
The Product Management System serves as a practical portfolio project that showcases the ability to design, develop, and deploy scalable business applications using modern web technologies.

The Product Management System is a full-stack web application designed to help businesses efficiently manage their product inventory, categories, and pricing. 
The application allows users to perform complete Create, Read, Update, and Delete (CRUD) operations on products and categories, making it easy to add new products, edit existing information, update stock quantities, and remove discontinued items. Each product contains detailed information such as its name, category, description, price, SKU.

**Technologies Used**
- Frontend: Angular
- Backend: ASP.NET Core Web API
- Database: SQL Server
- ORM: Entity Framework Core
- Language: TypeScript, C#, SQL
- Tools: Visual Studio, Visual Studio Code, Git, GitHub
